# Intel芯片，zsh终端，mac电脑 使用brew 清掉系统所有已安装的Java jdk 及配置

下面给出一条**“一键清掉 Intel Mac 上所有已安装的 Java（含 Homebrew、Oracle、AdoptOpenJDK、Zulu…）并还原成干净状态”**的完整命令链。  
**仅保留 zsh + Intel 芯片场景**，复制粘贴即可执行。

------------------------------------------------
1. 卸载所有 Homebrew 版 JDK
```bash
brew list | grep -i openjdk | xargs brew uninstall --ignore-dependencies
brew list | grep -i java   | xargs brew uninstall --ignore-dependencies
# 若还装过 Zulu、Oracle 等 cask：
brew list --cask | grep -iE '(java|zulu|oracle-jdk)' | xargs brew uninstall --cask
```

2. 手动扫尾删除（非 Homebrew）常见目录
```bash
sudo rm -rf \
  /Library/Java/JavaVirtualMachines/* \
  /Library/PreferencePanes/JavaControlPanel.prefPane \
  /Library/Internet\ Plug-Ins/JavaAppletPlugin.plugin \
  ~/Library/Application\ Support/Java \
  /usr/local/lib/java* \
  /usr/local/include/java* \
  /usr/local/share/java
```

3. 清掉以前写入的环境变量（zsh）
```bash
sed -i '' '/JAVA_HOME/d; /\/java/d' ~/.zshrc   # 删除旧行
unset JAVA_HOME
source ~/.zshrc
```

4. 验证是否归零
```bash
ls /Library/Java/JavaVirtualMachines || echo "✅ 已空"
which java || echo "✅ 找不到 java"
/usr/libexec/java_home -V 2>&1 | head -n 1
# 应输出 “Unable to find any JVMs…”
```

------------------------------------------------
执行完以上 4 步，系统就回到**“从未装过 Java”**的纯净状态。  
接下来想再装 Java 11，直接：
```bash
brew install openjdk@11
```
然后按之前给的配置写 `~/.zshrc` 即可。
