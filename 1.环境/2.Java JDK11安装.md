# Intel芯片，zsh终端，mac电脑使用brew安装Java 11

在 **Intel 芯片** 的 **macOS + zsh** 上，用 **Homebrew 安装 Java 11** 的完整流程（2025年验证）：

---

### ✅ 1. 安装 Java 11（OpenJDK）及确认安装
```bash
brew update

brew install openjdk@11

ls /usr/local/opt/openjdk@11/libexec/openjdk.jdk
```

---
### ✅ 2. 把 JDK 注册到系统可见位置（需要管理员密码，但不受 SIP 限制）
```bash
sudo ln -sfn /usr/local/opt/openjdk@11/libexec/openjdk.jdk \
             /Library/Java/JavaVirtualMachines/openjdk-11.jdk
```

---

### ✅ 3. 刷新环境变量（zsh 永久生效）
```bash
# 写入 zsh 配置文件 & 立即生效
export JAVA_HOME=$(/usr/libexec/java_home -v11)
echo "export JAVA_HOME=$(/usr/libexec/java_home -v11)" >> ~/.zshrc
source ~/.zshrc
```

---

### ✅ 4. 验证安装成功
```bash
/usr/libexec/java_home -V
# 输出应只有一行：
# 11.0.x (x86_64) "Homebrew" - "openjdk-11" ~/Library/...

java -version
# openjdk version "11.0.x"
```

---

### ✅ 5. 如果之前还装了别的版本，想强制只保留 11，可再加一句：
```bash
sudo rm -rf /Library/Java/JavaVirtualMachines/*jdk   # 先清掉旧注册
```
然后重复第 2 步即可。

---

### ✅ 6. 检查最终路径
```bash
which java
# → /usr/local/opt/openjdk@11/bin/java

echo $JAVA_HOME
# → /Library/Java/JavaVirtualMachines/openjdk-11.jdk/Contents/Home
```

---

✅ **完成！**你现在可以在 Intel Mac 的 zsh 终端中正常使用 `java` / `javac` 了。
