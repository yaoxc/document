### 一、快速生成子模块（以 gateway 为例）
1. 在父工程根目录 **右键 → New → Module → Spring Initializr**
    - **Artifact**: sc-gateway
    - **Packaging**: Jar
    - **Dependencies**:  
      – Spring Cloud Gateway  
      – Spring Boot Actuator  
      → **Finish**

2. IDEA 会自动把子模块加到父 pom `<modules>` 列表，子模块的 pom 里也会生成：

```xml
<parent>
  <groupId>com.example</groupId>
  <artifactId>sc-cloud-parent</artifactId>
  <version>1.0-SNAPSHOT</version>
</parent>
```

---

### 二、验证
1. 右侧 Maven 面板 **Reload** → 无红线即版本对齐成功
2. 启动子模块 `ScGatewayApplication` 控制台出现：
   ```
   Netty started on port 8080
   ```
   说明网关已跑通，父工程骨架完成。

### 六、常用 Cloud 子模块模板（后续继续 New Module 即可）
| 模块名 | 依赖勾选 |
|---|---|
| sc-gateway | Cloud Gateway |
| sc-user-service | Spring Web + Spring Data JPA |
| sc-config-server | Cloud Config Server |
| sc-nacos-registry | Cloud Bootstrap + Nacos Discovery（若用 Nacos） |
